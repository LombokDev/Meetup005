Deep Learning for Image Recognition

Deep Learning adalah salah satu bagian dari machine learning yang termasuk metode untuk bisa melatih jaringan saraf buatan multi lapis yang dapat mengenali pola data dengan sangat mendalam. Metode ini sangat efektif dan lebih mudah dalam mengidentifikasi pola dari data yang dimasukkan. Deep learning dirancang untuk terus menganalisis data dengan struktur logika yang mirip dengan bagaimana manusia mengambil keputusan. Aplikasi yang paling aplikatif dari deep learning adalah pengenalan gambar. Proses deep learning dimulai dari prapemrosesan data, perancanga arsitektur neural network, pelatihan, evaluasi model sampai dengan deployment. Untuk membangun arsitektur neural network dan melakukan pelatihan serta evaluasi model, dapat menggunakan framework deep learning seperti KERAS yang sangat cocok untuk pemula. Materi yang diusukan sebagian besar terdiri atas: pengenalan neural network (deep learning) dan demo. Adapun kasus image recognition yang diusulkan untuk demo adalah pengenalan gambar hewan (cat or dog or panda) yang akan dilatih dengan arsitektur neural network yang paling sederhana dari tahap preprocessing sampai dengan deployment.

Berkenalan Dengan VueJS

VueJs merupakan progresif framework javascript yang digunakan untuk mengembangkan antarmuka web interaktif. VueJs berfokus pada bagian view atau front end sebuah web. VueJs sangat mudah di integrasikan dengan project dan library lainnya. VueJS dibuat oleh Evan You, mantan front-end developer dari Google.
